# Changelog

Todos los cambios importantes de este proyecto se documentan en este archivo.

El formato sigue las recomendaciones de **Keep a Changelog** y el versionado semántico (**Semantic Versioning**).

---

## [1.0.0] - 2026-07-17

### Added

- Cálculo de hashes MD5, SHA-1 y SHA-256.
- Lectura eficiente de archivos por bloques.
- Barra de progreso durante el cálculo.
- Ejecución en segundo plano mediante hilos.
- Cancelación segura del cálculo.
- Verificación de integridad mediante comparación de hashes.
- Copia del hash al portapapeles.
- Soporte para arrastrar y soltar archivos (Drag & Drop).
- Gestión robusta de errores.
- Arquitectura modular basada en UI, Controller y Hashing.
- Pruebas automatizadas con pytest.
- Empaquetado mediante PyInstaller.
- Sistema de recursos e iconos para macOS y Windows.
